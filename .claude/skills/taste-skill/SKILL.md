# Taste Skill — Good Design Judgment

## Core Principle
Every visual decision must feel INTENTIONAL. No defaults. No generic choices.

## Typography Rules
- Never default to Inter, Roboto, or Arial
- Always pair a personality font (display) with a readable font (body)
- Use unexpected weights: ultralight titles with heavy accents, or vice versa
- Letter-spacing matters: tight for impact, wide for elegance

## Color Rules
- Choose a dominant color and commit to it fully
- Max 2 accent colors — one warm, one neutral
- Dark mode: never pure black (#000). Use #0a0a0a, #0d0d0d, #111
- Light mode: never pure white (#fff). Use #fafafa, #f5f5f0, #faf8f5

## Spacing Rules
- Sections need breathing room: minimum 80px padding top/bottom
- Between title and content: at least 24px
- Cards should never feel claustrophobic — generous internal padding

## Component Rules
- Buttons: never fully rounded unless the brand is playful. 4-8px radius for professional.
- Cards: subtle shadows over harsh borders
- Images: always with aspect-ratio locked. Never stretch or squish.
- Icons: consistent size and weight throughout

## What to AVOID
- Generic hero with centered text + stock photo
- Blue CTA buttons by default
- Card grids that look like every other website
- Animations that serve no purpose
